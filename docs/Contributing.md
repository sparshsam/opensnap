# Contributing

OpenSnap is a personal project. Issues and pull requests are welcome.

## Guidelines

- Open an issue first to discuss changes before opening a PR.
- Keep the widget philosophy in mind: minimal, focused, fast.
- New features should justify their UI complexity.
- Use fully qualified names for types that are ambiguous with `System.Drawing` (`Brush`, `Color`, `MouseEventArgs`, etc.).
