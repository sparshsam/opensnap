# Changelog

## v1.0.0 — Stable Release (Jun 2026)

- Repo made public
- Stable branch, issue templates, SECURITY.md, CODE_OF_CONDUCT.md
- Full contributor and developer documentation
- QA checklist, FAQ, troubleshooting guide
- MSIX packaging, Partner Center-ready
- Everything from v0.8 through v0.9.5 included and finalised

## v0.9.5 — Accessibility & Localization (Jun 2026)

- Keyboard navigation (arrow keys + Enter/Space)
- Screen reader labels (AutomationProperties.Name)
- High Contrast mode support
- Larger icons toggle
- 5-language localization (EN, FR, DE, ES, JA)
- Language picker in Settings

## v0.9.0 — Distribution & Enterprise (Jun 2026)

- Per-user/per-machine installer, silent install support
- Background update download + one-click install
- Changelog viewer, diagnostics page, log export
- "Check for updates" tray menu item
- MSIX packaging config

## v0.8.0 — Workflow Power Tools (Jun 2026)

- Post-capture quick actions popup (Copy, OCR, Paint, Reveal)
- Project prefix `{prefix}` and sequential numbering `{seq}`
- Date-based subfolders (yyyy/MM-yyyy)
- Pin/delete/clear screenshot history
- Custom app path for "Open with..."
- Settings UI for naming and custom app

## v0.7.0 — Triple-button pill (Jun 2026)

- 240px pill with 3 clickable sections (area toggle, full screen, active window)
- Spring-back bounce animation per section
- Green border flash (#007a3f)
- Per-section hover glow, glass dividers
- Magnetic edge snapping
- Opacity slider (20–100%)
- Auto-hide on fullscreen apps
- DPI awareness (PerMonitorV2)
- Multi-monitor area selection fix
- Exception logging

## v0.6.0 — Installer & distribution (Mar 2025)

- Inno Setup installer, Start Menu + desktop shortcut
- About dialog (version, GitHub link, license)
- Global hotkey configuration in Settings
- Area selection fix, active window fix, settings crash fix
- Installer preserves settings on upgrade
